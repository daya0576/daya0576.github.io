# Rain Effect - About Page

## Overview
Add a rain atmosphere effect to the About page with visual rain, ripple animation, and ambient sound.

## Components

### 1. Rain (Top)
- **File:** `layouts/partials/rain.html`
- Animated raindrops falling from the top of the page

### 2. Ripple (Bottom)
- **File:** `layouts/partials/ripple.html`
- Water ripple effect at the bottom where raindrops land

### 3. Rain Music
- **File:** `static/rain.mp3`
- Ambient rain sound, user-triggered playback
