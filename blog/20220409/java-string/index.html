关于 Java 字符串的秘密

发表于 2022-04-09 分类于 JAVA

最近对 java 字符串（java.lang.String）的部分行为感到困惑，抽空查阅资料后豁然开朗。忍不住写一篇博客纪念一下

一、不变性

从源码中不难看出：

  • class 用 final 修饰：不能被继承 & override
  • value 变量是 final + private 的：一旦被赋值，无法被更改内存地址（禁止重新赋值），同时外部无法访问内部数组进行修改。
